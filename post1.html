DEXTER Voice Assistant

March 30, 2023


I recently started working on a Voice Assistant using Python as the primary programming language. Now you might be asking "Why are you creating your
own Voice Assistant?" Recently, I started getting into hackathons, which is a type of competetion to see who can make the best technological project
in a certin period of time. I am going to be using this voice assistant for the upcoming Monclair University Hackathon. My team and I have not came
up with a name for him yet, so we decided on the name "DEXTER" for now. DEXTER is going to be very similar to "OpenSmartHome", which is a previous
hackathon project winner from the 2022 Rowan University Hackathon. DEXTER is going to have similar API's and overall similar syntax. Obvoiusly, we are not
going to copy the code over, we are going to write our own and base it off of OpenSmartHome because, why reinvent the wheel.
